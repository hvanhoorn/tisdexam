

DOCUMENT CLASS tisdexam
=======================

versie: 2.0
datum:  12-sep-2017


-----------------------
Dit is de nieuwe document class tisdexam die geschikt is voor de
toetsvoorbladen vanaf het schooljaar 2017-2018.

Zowel de nederlandse als de engelse toetsvoorbladen zijn geimplementeerd.

Er is zoveel gewijzigd ten opzichte van het vorige toetsvoorblad zodat er
geen poging is gedaan om de oude opties en commando's af te vangen met
een foutmelding. De document class is tamelijk groot, zo'n 2 MB. Dat komt
omdat de logo's van TIS erg veel ruimte innemem.

LEES VOORAL DE DOCUMENTATIE.


Deze document class is getest met TeXlive 2017 en TeXmaker 4.5.


Jesse, 12-sep-2017
